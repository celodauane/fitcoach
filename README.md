# Sunday

Your AI fit coach. Personalized 12-week weight loss programs.

## Features

- Science-based calculations (Mifflin-St Jeor BMR, TDEE)
- Personalized cardio progressions
- Macro targets with sustainable deficits
- Adherence strategies included
- Safety rails built-in

## Tech Stack

- Cloudflare Workers
- Workers AI (Llama 3.1 70B)
- Vanilla HTML/CSS/JS
- Linear-inspired monochrome design

## Deploy

```bash
npm install
npx wrangler deploy
```

## Local Development

```bash
npm install
npm run dev
```
